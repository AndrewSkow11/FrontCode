Селекторы потомков

Селекторы потомков позволяют выбрать все элементы, которые находятся внутри другого элемента, независимо от их уровня вложенности. Необходимо написать родительский элемент, затем пробел и этот элемент, который нужно выбрать

Селекторы дочерних элементов

Селекторы дочерних элементов child выбирает только прямых потомков родительского элемента (испольузется символ > между родительским и дочерним элементами).

Соседние селекторы

Соседние селекторы - это суперточный инструмент, который позволяет выбрать только те элементы, которые находятся срузу после указанного элемента.
Пример: есть список, в котором должен выделиться только первый пункт, остальные - нет. Для использования соседнего селектора нужно соедить первый элемент, добавить символ +, а затем указать второй элемент
