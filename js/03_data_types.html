Типы данных

Основыне типы данных

  1. Строки string
  2. Числа number
  3. Логические значения boolean
  4. Undefined - переменная, которой не присвоено значение
  5. Null - специальное значение, означающее ничего

Строки

Строки представляют собой последовательность символов, заключенных в двойные или одинарные кавычки. Например: "Hello, World!"
Строки можно объединять при помощи символа +

Числа (number)

Числа представляют собой целые и дробные числа. Например: 123, 45.67
Можно использовать разные разделители десятичной части, например: 123.456, 123,456.789

Булевы переменные

Undefined

Undefined - это специальное значение, которое означает, что переменная не присвоено никакого значения. Например: let myVariable;
В этом случае myVariable будет равно undefined

Null

Null - это специальное значение, которое означает, что переменная не имеет никакого значения. Например: let myVariable = null;
В этом случае myVariable будет равно null
