Работа с DOM

Document Object Model (DOM) - это спецификация, описывающая структуру и поведение объектов документа в веб-браузере. DOM предоставляет доступ к веб-странице как к дереву объектов.

С помощью DOM можно:

  1. Управлять элементами страницы: добавлять, удалять и изменять их
  2. Делать страницы живыми: кнопки, которые реагируют на клики, анимациии, появляющиеся сообщения
  3. Создавать сайты, которые не просто статичны, а взаимодейтсвуют с пользователем

DOM - это интерфейс, который позволяет JS общаться с HTML - страницей. Браузер читает HTML и создаёт из него дерево объектов. Каждый тег становится узлом. Вложенные элементы имеют детско-родительские отношения)).
DOM - это не HTML, а это модель страницы, которую браузер создаёт для работы с ней.

Основная цель DOM - дать доступ к элементам страницы с помощью JavaScript

Корень дерева - <html>
Внутри находятся <head> и <body>
Остальные элементы - ветки и листья
