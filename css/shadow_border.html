Добавление границ и теней

CSS позволяет добавлять границы и теней к элементам.

Это элемент с добавленными границами и тенью.

border: 5px dashed red

border: 5px dotted red

border-top:3px solid green; верхняя граница
border-right: 8px dashed blue; правая граница
border-bottom: 7px dotted violet; нижняя граница
border-left: 2px solid navy; левая граница

Параметры box-shadow

.with_shadow{ width: 200px; height: 100px; background-color: lightblue; box-shadow: 10px 10px 15px rgba(0, 0, 0, 0.5); - тень сдвинута на 10 px вправо и вниз, 15px - мягкость и размытие краям тени rgba(0, 0, 0, 0.5) - чёрный цвет с прозрачность 0.5 }
