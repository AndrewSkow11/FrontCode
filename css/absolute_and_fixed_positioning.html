Ключевые значения свойства position

  1. static (по умолчанию) - элемент находится там, где его разместил HTML
  2. absolute - элемент оторван от потока документа и привязан к своему позиционированному родителю
  3. fixed - элемент приклеивается к окну браузера и остаётся там даже при скролинге страницы
