Свойство transform

Наведи на меня scale(1.2)

примеры transform

  • transform: scale(1.2) - увеличение элемента на 20%
  • transform: rotate(45deg) - поворот на 45 градусов
  • transform: translate(50px, 100px) - перемещение на 50px вправо и 100px вверх
  • transform: skew(30deg, 15deg) - наклонение на 30 градусов вправо и 15 градусов вниз

@keyframes

@keyframes - для создания сложных анимаций, которые могут быть использованы в нескольких местах вашего CSS
animation-name - название анимации
animation-duration - продолжительность анимации в секундах
animation-timing-function - способ изменения скорости анимации (linear, ease, ease-in, ease-out, ease-in-out)
animation-delay - задержка в начале анимации в секундах

Анимация движения

Анимация может быть: линейной linear,
плавной ease,
с ускорением ease-in
или замедлением ease-out

итог

transition - Плавно изменяет свойства элемента.
transform - Изменяет внешний вид элемента (размер, поворот, положение).
