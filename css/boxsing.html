Box-sizing и управление размерами

box-sizing - это способ, которым браузер рассчитывает общие размеры элемента

По умолчанию width, height элемента включают в себя только содержимое content.

В случае добавления отступов padding или границы border блоки прибавляются к ширине и высоте.

div { width: 100px; padding: 20px; border: 5px solid black; }

свойство box-sizing

  • content-box - старый режим, включает в себя width, height, padding и border
  • border-box - новый режим, включает в себя width, height, padding, border и margin
div.new { width: 100px; padding: 20px; border: 5px solid black; box-sizing: border-box; }

max min width

div.minmax { width: 50%; max-width: 300px; min-width: 100px; }
