Каскадность и наследование

Каскадность - это основной приницип CSS, который определяет какое правило будет применено к элементу, если у него несколько конфликтующих стилей.

Браузер выбирает стили по приоритету:

  1. Инлайновые стили (атрибут style) имеют самый высокий приоритет
  2. Внутренние стили (тег style) идут вторыми
  3. Стили из файла CSS (ссылки link или @import) идут третьими - наименьший приоритет

Не все свойства предеаются автоматически по наследству

Не наследуется фон элемента background, отступы padding, margin. Если нужно наследование, то оно указывается явно с помощью inherit

Специфичность: чем конкретнее селектор, тем выше его приоритет

  1. id (#) - самый высокий приоритет
  2. классы (.) - средний приоритет
  3. элементы (element) - низший приоритет

Когда два правила имеют одинаковую специфичность, брузер применяет то правило, которое было написано последним. Слово !important даёт стили наивысший приоритет.
