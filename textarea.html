Textarea

<textarea> - поле ввода длинного текста. Он позволяет ввести много текста с возможностью выделения.
Textarea требует закрывающего тега.

Атрибуты <textarea>

  1. rows: задаёт количество строк, то есть высоту поля
  2. cals: задаёт количество символов в строк (ширина поля)


Атрибут maxlength <textarea> ставит лимит на количество символов, которые можно ввести в поле. Если введенное число превышает лимит, поле автоматически обрезается до указанного количества символов.

Как только лимит будет исчерпан, больше вводить будет нельзя.
